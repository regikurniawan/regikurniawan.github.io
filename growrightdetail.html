Project Details

GrowRight adalah aplikasi Android natif yang dikembangkan menggunakan Kotlin, dirancang untuk membantu pengguna dalam memantau pertumbuhan anak dan menilai potensi risiko stunting.

Project information

  • Kategori: Mobile App
  • Tema: Kesehatan
  • Tanggal Proyek: Juni 2024
  • Github: Grow-Right

Penjelasan Aplikasi

Ringkasan Proyek:
  • GrowRight adalah aplikasi Android natif yang dikembangkan menggunakan Kotlin, dirancang untuk membantu pengguna (kemungkinan orang tua atau pengasuh) dalam memantau pertumbuhan anak dan menilai potensi risiko stunting.
  • Aplikasi ini terintegrasi dengan API backend untuk melakukan analisis stunting berdasarkan data anak yang dimasukkan pengguna.
  • Berfungsi sebagai alat kesadaran dini dan menyediakan akses ke informasi kesehatan yang relevan, sejalan dengan tujuan pemberdayaan kesehatan dan kesejahteraan di komunitas rentan.

    • Fitur Utama
  • Autentikasi Pengguna: Sistem registrasi (Username, Email, Password) dan login pengguna yang aman , berinteraksi dengan API backend.
  • Manajemen Sesi: Menggunakan SharedPreferences (ManageSession) untuk mengelola sesi login pengguna dan token autentikasi.
  • Dasbor Utama (Home): Pusat navigasi utama yang menampilkan sapaan personal ("Hai, [Username]").
  • Menyediakan akses cepat melalui ImageButton ke: Informasi Stunting (melalui Web Intent), Informasi Kebutuhan Nutrisi (melalui Web Intent), Fitur Analisis Stunting, Profil Pengguna dan Riwayat Analisis.
  • Input Analisis Stunting: Layar khusus (AnalyzeActivity) bagi pengguna untuk memasukkan data anak: Nama, Jenis Kelamin (Male/Female), Tinggi (Cm), Berat (Kg), dan Usia (Bulan). Menggunakan Custom Views (EditTextGender) untuk validasi input.
  • Integrasi API untuk Prediksi: Mengirim data anak yang terkumpul ke endpoint API backend (ApiService.getPrediction) untuk menerima hasil analisis stunting.
  • Tampilan Hasil: Menyajikan hasil analisis (ResultActivity), menampilkan data input dan skor prediksi (Z-score untuk Tinggi-Usia, Berat-Usia, Berat-Tinggi) beserta persentase risiko yang dihitung . Menyertakan interpretasi tekstual hasil (misalnya, Tidak Stunting, Stunting, Stunting Parah) berdasarkan persentase.
  • Penyimpanan Riwayat Lokal: Menyimpan hasil prediksi secara lokal menggunakan Room Database (AppDatabase, PredictionDao, PredictionDB) memungkinkan pengguna melacak analisis sebelumnya.
  • Tampilan Riwayat: Menampilkan riwayat prediksi yang disimpan (kemungkinan menggunakan RecyclerView dan HistoryAdapter) untuk ditinjau.
  • Manajemen Profil: Memungkinkan pengguna melihat informasi profil mereka (Username, Email) yang diambil dari API . Menyediakan opsi untuk mengedit detail profil (EditProfileActivity) dan logout.
  • Edit Profil: Memungkinkan pengguna memperbarui Username, Email, dan Nomor Telepon mereka melalui panggilan API (ApiService.updateProfile).
