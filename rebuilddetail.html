Project Details

Aplikasi Rebuild adalah aplikasi pengelolaan sampah mobile Android natif yang dibangun menggunakan Kotlin.

Project information

  • Kategori: Mobile App
  • Tema: Daur Ulang dan Pembuangan Sampah
  • Tanggal Proyek: Januari 2024

Penjelasan Aplikasi

Ringkasan Proyek:
  • Aplikasi mobile Android natif (dibangun dengan Kotlin) yang berfungsi sebagai alat bantu pengelolaan sampah pribadi.
  • Menggunakan arsitektur Single-Activity dengan Fragments untuk navigasi.
  • Memungkinkan pengguna untuk mencatat (log) sampah yang mereka daur ulang dengan mengambil foto dan memasukkan berat.
  • Menerapkan komponen Android Jetpack (Room, ViewModel, LiveData) untuk mengelola dan menampilkan riwayat data sampah secara persisten dan efisien.

    • Fitur Utama
  • Autentikasi Pengguna: Sistem registrasi (Username, Email, Password) dan login pengguna. Data kredensial pengguna disimpan secara lokal menggunakan database SQLite (SQLiteOpenHelper).
  • Manajemen Sesi: Menggunakan SharedPreferences untuk menyimpan sesi login pengguna (email dan nama).
  • Navigasi Berbasis Fragment: Menggunakan BottomNavigationView untuk beralih antar 4 Halaman (Fragment) utama: Home, Trash (Input), History, dan Profile.
  • Halaman Utama (Dashboard): Menyapa pengguna secara personal ("Hai, [Nama]").
  • Menyediakan tautan cepat menggunakan Implicit Intent untuk: Membuka Google Maps dan mencari "Bank Sampah" terdekat, membuka halaman web eksternal untuk tips daur ulang dan informasi.
  • Input Data Sampah (Fragment "Trash"): Memungkinkan pengguna memilih gambar sampah dari galeri perangkat (menggunakan ActivityResultContracts), memasukkan berat sampah dalam kilogram (kg) dan menyimpan data ini (URI gambar dan berat) ke dalam database Room.
  • Riwayat (Fragment "History"): Menampilkan daftar semua sampah yang telah di-submit pengguna dalam sebuah RecyclerView, setiap item menampilkan preview gambar dan catatan beratnya dan memiliki fungsi "Hapus" pada setiap item untuk menghapus data dari database Room.
  • Halaman Profil: Menampilkan nama dan email pengguna yang sedang login (diambil dari SharedPreferences) serta menyediakan tombol "Logout" yang menghapus data sesi dari SharedPreferences dan mengembalikan pengguna ke halaman Login.
